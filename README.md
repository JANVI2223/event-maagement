# 🎉 Event Management Website

A modern and responsive **Event Management Website** developed using **HTML, CSS, and Bootstrap**. This project is designed to provide users with an easy and attractive platform to explore events, learn about services, and get in touch with the event management team.

## 📌 About the Project

The Event Management Website is a front-end web development project created using the **Bootstrap framework**. It provides a clean, responsive, and user-friendly interface for an event management company.

The website can be used to showcase different types of events such as weddings, birthday parties, corporate events, concerts, and other special occasions.

## ✨ Features

- 🏠 Attractive Home Page
- 📅 Events and Services Section
- 💍 Wedding Event Management
- 🎂 Birthday Party Management
- 🏢 Corporate Event Management
- 🎵 Concert and Party Events
- 📸 Gallery Section
- 📞 Contact Us Section
- 📱 Fully Responsive Design
- 🎨 Modern UI using Bootstrap
- 🧭 Responsive Navigation Bar
- 🔘 Bootstrap Buttons and Cards
- 📋 Event Registration/Booking Interface

## 🛠️ Technologies Used

- **HTML5** – For creating the structure of the website
- **CSS3** – For custom styling and design
- **Bootstrap 5** – For responsive layout and UI components
- **JavaScript** – For basic interactivity
